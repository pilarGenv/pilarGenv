🥇 Gold Recovery Prediction

Context

Mining companies need accurate predictions of gold recovery to optimize production efficiency and reduce operational costs. This project uses machine learning to predict the gold recovery rate from industrial process data.

Analysis

The project involved data validation, preprocessing, exploratory data analysis, outlier detection, feature evaluation, and the comparison of multiple machine learning models using cross-validation and the sMAPE evaluation metric.

Key Conclusions

The selected machine learning model achieved the best predictive performance for estimating gold recovery, demonstrating how data science can support decision-making and process optimization in the mining industry.

Technologies

Python • Pandas • NumPy • Scikit-Learn • Matplotlib • Machine Learning
